Two different variants of the sticky notebook. Sticky_notebook.py is a simple sticky-like program.
Simply made with tkinter to add, remove and clear notes with persistent data.
	- tkinter is in the standard library so no additional installation is required

The second variant is made  with customtkinter which is more customizable, same principal except it is better looking and has an implemented Dark and Light mode which the user can switch between.
	- In order to use the second variant with customtkinter you need to install customtkinter with
	- "pip install customtkinter"


Both programs are very well commented (might even be over commented) to make everything crystal clear for everyone.

Assignment did say only use what we learn us in our course but I had some time over so I decided I wanted to make something that looked better than with normal tkinter, so I made a second version with customtkinter.
